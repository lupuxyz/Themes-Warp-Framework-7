主题设置

一般设置里，主题允许你控制它的表现形式和整体的外观。

样式

在样式部分，包含了定制工具（Customizer）。点击 定制工具（Customizer） 按钮添加或修改现有的样式。定制工具打开以后，可以轻松地定制色彩、字体、尺寸等等样式来修改主题。定制工具只会保持被修改的变量，并将它们编译到CSS中。

需要了解更多信息，请查看定制工具使用教程。

开发

1. LESS 开发者模式

查看第一个选项，每当加载一个页面时，通过JavaScript使浏览器自动编译LESS文件（browser automatically compiles your LESS files） 。这提供了一个合适的测试环境。不推荐在已上线的网站启用开发者模式，因为编译过程的效率是非常低的。

2. 动态样式

第二个选项，允许浏览器通过RL加载样式（load styles by URL）。如果想要测试网站的不同样式，这会很有用。网站上线后，需要禁用此项。

3. 将LESS编译成CSS

点击 编译LESS（Compile LESS） 按钮，完成LESS的修改，你所有的修改都会被编译成为最终的CSS。

重要 必须确保你没有在 /styles/STYLE-NAME/css 文件夹中进行开发，因为这些文件夹会在点击编译LESS（Compile LESS） 按钮时立即被覆写。

压缩

在下列压缩选项中选择一个，以优化页面加载时间。

设置 描述
None 所有文件都将以分开并且完整体积的状态被加载。
Combination + Minify 此选项将会最小化地将CSS和JavaScript合并到一个单独的文件中。
Combination + Minify + Data URIs Data URI 直接将图像数据嵌入文档中，从而减少了图像文件的HTTP请求。这仅适用于最大10KB的图像。
Combination + Minify + Data URIs + Gzip Gzip 允许通过压缩数据包实现更快的数据传输，替代原有的通过分别进行传输来处理大数据量的方式。

响应式

视口meta标签控制移动设备浏览器中的布局。如果没有设置它，手机上的默认宽度是980px，并且网站会看起来和PC浏览器中一样。

Bootstrap (只针对Joomla)

选择此项，禁用 Bootstrap。如果你仅仅使用Joomla的博客与文章，这将会减少页面加载时间。

文章 (只针对WordPress)

在页面中设置显示1列或最多4列布局样式，并设定文章的排列顺序是横向排列还是纵向排列。另外，选择 wpautop 选项，启用自动为段落添加HTML标签的功能。

内容

启用或禁用回到顶部与Warp标志

社交按钮

选中Social buttons选项，让用户可以将文章分享给他人。它会在文章下面插入小工具。这些都是通过来自Twitter, Facebook 或 Google+的JavaScript进行加载的。

小工具/模块（Widgets/Modules）

为特定位置上的默认小工具和模块设置样式。

注意 在小工具/模块设置中，一旦为某个单独的小工具/模块选择了样式，此处设定的样式将立即被覆写。

附加的脚本

你可以在这里添加谷歌统计代码等附加的脚本。它会被添加在body元素结束标签前。

  
var _gaq = _gaq || [];
  _gaq.push(['_setAccount', 'UA-XXXXX-X']);
  _gaq.push(['_trackPageview']);

  (function() {
    var ga = document.createElement('script'); ga.type = 'text/javascript'; ga.async = true;
    ga.src = ('https:' == document.location.protocol ? 'https://ssl' : 'http://www') + '.google-analytics.com/ga.js';
    var s = document.getElementsByTagName('script')[0]; s.parentNode.insertBefore(ga, s);
  })();
