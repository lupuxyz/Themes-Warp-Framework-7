小工具/模块 设置

管理Wordpress小工具与Joomla模块的样式和其他设置。

小工具或模块可以以不同的样式显示，还能加入图标和徽章。在Warp的小工具/模块总览中，小工具与模块都根据它被安排的位置进行分组。另外，还可以通过关键词和指定的位置区分它们。

设置

下面的表格给出所有可用的小工具/模块设置项的总览：

设置项 描述
Class 允许添加自定义的CSS类。
样式 在选择框中选择一个预设的小工具/模块样式。
图标 允许你从图标预览中选择一枚图标。还可以输入路径以显示自定义图片。在Joomla中，输入完整的路径，如： /images/MY-IMAGE.jpg。WordPress默认选择/wp-content/uploads文件夹，所以路径只需要这样就行，如：MY-IMAGE.jpg。
徽章 为小工具/模块添加一枚徽章。输入徽章的文本并选择一种色彩。
显示 在PC、平板电脑或手机中，某一特定设备上显示或隐藏。

针对Wordpress的设置

设置项 描述
标题（Title） 显示或隐藏小工具/模块的标题。
分配（Assignment） 通过将小工具/模块指派到菜单项来规定在哪些页面上显示这些小工具/模块。可以通过按住Ctrl键选定多项。

注意 还可以添加新的小工具/模块样式。或创建新的位置。了解更多信息，见小工具/模块文档。
