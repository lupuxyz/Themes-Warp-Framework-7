禁用响应式布局

学习如何通过关闭视口meta标签和编辑断点来禁用响应式布局。

1. 视口meta标签

在主题设置中的 设置 -> 响应式 菜单里，选中禁用视口meta标签/Disable the viewport meta tag选择框。这样将使得移动设备上网页的布局宽度默认为980px，整个页面看起来和桌面设备上一样。

2. 断点

下一步是给断点设置最小值和最大值，避免布局的响应式行为。为此，打开定制工具并设置Mini, Small, Medium 和 Large 断点数值为0px。而最大的Xlarge断点数值越大越好，比如9999px。这样便能确保浏览器不再响应式断点。

3. Less

你需要添加以下CSS代码到/less/theme.less文件。

.uk-container {
    max-width: none;
    width: @utility-container-max-width;
}

注意查看自定义章节，学习如何覆写文件。
