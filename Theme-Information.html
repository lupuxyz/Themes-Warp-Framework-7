主题信息

获取关于主题的版本、文件修改以及潜在问题的概览。

版本/Versions

仪表盘中显示主题和框架当前的版本号。这些信息在需要解决技术问题或更新主题与框架是是非常有用的。

系统检查/System Check

系统检查显示所有重要的文件或潜在的问题。

问题 描述
文件权限 提供一般的信息，见文件权限问题。
PHP 版本 检查主题是否运行在未支持或已弃用的PHP版本中。
PHP 扩展 检查必要的PHP扩展是否已安装：
JSON
DOM XML
Multibyte string (mbstring)

文件验证

文件验证帮助你轻松追踪定制行为。它无限期地显示所有已修改和已删除的文件。

注意 如果FTP服务器的传输模式设置为文本模式，文件会自动被格式化。因此你自己的修改可能不再被识别。为了避免这种情况，设置传输模式为二进制。
